# Nodejs-real-time-chat-app

### Basic real time chat application with: 
* Node.js 
* socket.io
* ExpressJS

### Install Dependencies

```console
npm install
```

### Running app

```console
npm start
```

### Can expand later by adding the following features:
* Login form
